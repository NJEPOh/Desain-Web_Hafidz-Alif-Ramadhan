Contoh Penggunaan Hyperlink (<a>)

1. Atribut href

  • Link ke WWW Server
  • Link ke FTP Server
  • Link ke Gopher Server
  • Link ke Email
  • Link ke Newsgroup
  • Link ke bagian tertentu dalam halaman (anchor)

2. Atribut target

  • Buka di window yang sama (_self)
  • Buka di window induk (_parent)
  • Buka mengganti seluruh isi window (_top)
  • Buka di tab/window baru (_blank)
